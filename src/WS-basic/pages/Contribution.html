Contribution

Protease database

To capture and inactivate COVID-19, we focus on the detroy of the spikes on virus. Thus, we’ve created a protease database searching software aming to find the best protease for SARS COV2. As decribed, this database is not only for SARS COV2, but other virus could use as well, when there is other virus outbreak, we could find the relative protease more efficient.

Detection box

We also invented a 3D printed detection box which is portable and have fast virus detection, instead of using PCR. This way, we could help the government to do the primary screening at the airport. For this detection box, we could not only target sars cov2 but also other virus, such as flu virus. The only change need to make is just the receptor of the virus on the gel.

To sum up, both our protease database and our 3D detection box can be widely used.
